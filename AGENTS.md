# Repo Context

This repository packages a finance-oriented literature review workflow for multiple coding agents.

## Working Rules

- Keep provider-native files aligned when the core workflow changes.
- Prioritize top finance journals first, then adjacent top economics and accounting journals, then working papers.
- Always distinguish published papers from working papers.
- Never fabricate citations, metadata, or journal quality signals.
- For empirical finance papers, capture:
  - research question,
  - sample,
  - period,
  - unit of observation,
  - identification strategy,
  - main result,
  - mechanism,
  - main limitation,
  - venue tier.
- Default outputs should include:
  - scoped question,
  - search strategy,
  - prioritized reading list,
  - literature matrix,
  - synthesis by theme and design,
  - explicit research-gap statement.
