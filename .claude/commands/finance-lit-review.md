## Workflow

1. Scope the question tightly.
2. Set the venue ladder:
   - top finance journals first,
   - then adjacent economics or accounting journals if relevant,
   - then frontier working papers from NBER, CEPR, SSRN, and RePEc / IDEAS.
3. Collect and verify citations.
4. Extract a finance matrix for the important papers:
   - citation,
   - venue tier,
   - published vs working paper,
   - research question,
   - sample / setting,
   - period,
   - unit of observation,
   - identification strategy,
   - main finding,
   - mechanism,
   - main limitation.
5. Synthesize by mechanism and design, not paper-by-paper unless the user explicitly asks for that.
6. Deliver:
   - scope note,
   - prioritized reading list,
   - literature matrix,
   - synthesis,
   - gaps,
   - verified citation block.

## Hard Rules

- Never fabricate citations.
- Always label working papers as working papers.
- Say clearly if only titles or abstracts were available.
- Do not let working papers crowd out established published evidence in mature literatures.
