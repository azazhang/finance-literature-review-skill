# Finance Literature Review Workflow

Use this workflow when the user asks for:

- a finance literature review,
- a related literature section,
- an annotated bibliography,
- a finance reading list,
- a working-paper scan,
- a gap map across finance, economics, and accounting.

## Core Rules

- Start with top finance journals.
- Widen to top economics journals only when the question genuinely overlaps.
- Widen to top accounting journals when the topic overlaps disclosure, reporting, auditing, governance, or information environment.
- Separate published papers from working papers.
- Never fabricate citations or metadata.
- If only titles or abstracts are available, say so directly.

## Extraction Checklist

For each core empirical paper, record:

- citation,
- venue tier,
- published or working paper,
- research question,
- sample / setting,
- sample period,
- unit of observation,
- identification strategy,
- main finding,
- mechanism,
- main limitation.

## Default Deliverables

- scope note,
- prioritized reading list,
- literature matrix,
- synthesis by mechanism and design,
- explicit research gaps,
- verified citation block.
