# Output Modes

## Reading List

Use when the user wants fast orientation.

Return:

- 10-20 prioritized papers,
- a short reason each paper matters,
- separation of published vs working paper.

## Literature Matrix

Use when the user wants comparison across papers.

Return the matrix with:

- venue tier,
- design,
- identification strategy,
- mechanism,
- limitation.

## Related Literature Section

Use when the user wants prose suitable for a paper or dissertation.

Organize by:

- mechanism,
- identification strategy,
- setting,
- agreement vs disagreement in findings.

## Gap Map

Use when the user is searching for project ideas.

Return:

- saturated questions,
- unresolved questions,
- under-identified areas,
- under-studied settings,
- frontier but provisional areas.
