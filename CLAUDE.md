# Finance Literature Review Context

Use the finance literature review workflow when the user asks for:

- a literature review,
- related literature section,
- annotated bibliography,
- gap map,
- journal-targeted reading list,
- working-paper scan,
- finance / economics / accounting research synthesis.

## Default Priorities

- Finance first: `Journal of Finance`, `Journal of Financial Economics`, `Review of Financial Studies`.
- Then adjacent top economics and accounting journals when they are genuinely relevant.
- Then frontier working papers from NBER, CEPR, SSRN, and RePEc / IDEAS.

## Hard Rules

- Never hallucinate citations.
- Always label working papers as working papers.
- Prefer synthesis by mechanism and identification strategy, not paper-by-paper summary.
- Capture sample period, unit of observation, and main empirical design for each core paper.
- If only titles or abstracts are available, say so explicitly.

## Preferred Outputs

- Scoped question and boundaries.
- Venue-prioritized reading list.
- Literature matrix.
- Thematic synthesis.
- Explicit gaps and next-step reading suggestions.

Use `/finance-lit-review` for the explicit guided workflow.
